# Honey-Finder

미국 주식 투자 후보를 미너비니식 기술 조건, 시가총액, 거래대금, 간단한 재무 품질 지표로 진단하는 디스코드 봇 MVP입니다.

이 봇은 단순히 "매수할 종목"을 찍어주는 도구가 아니라, 관심 종목군 안에서 차트상 강하고 거래 가능한 후보를 선별하고, 추천 이후 실제 성과가 어땠는지까지 기록해 검증하는 의사결정 보조 도구입니다.

## 핵심 명령어

### `/미너비니 종목추천`

`.env`의 `DEFAULT_TICKERS`에 들어 있는 미국 주식 후보군을 훑고, 미너비니 조건 점수와 검증 기준을 통과한 종목을 후보로 보여줍니다.

추천 결과는 자동으로 `data/recommendations.json`에 저장됩니다. 이 파일은 개인 기록이므로 GitHub에는 올라가지 않도록 `.gitignore`에 포함되어 있습니다.

예시:

```text
/미너비니 종목추천
/미너비니 종목추천 개수:10 최소점수:70
```

### `/미너비니 진단`

특정 티커를 개별 진단하고, 리포트와 차트 이미지를 디스코드에 전송합니다.

```text
/미너비니 진단 티커:NVDA
```

### `/미너비니 검증`

추천 후보가 현재 기준을 통과하는지 확인하는 사전 검증 명령입니다.

```text
/미너비니 검증
/미너비니 검증 티커:NVDA
/미너비니 검증 티커:NVDA 최소점수:70
```

검증 기준:

- 가격 데이터가 충분한가
- 최소 점수를 넘었는가
- 핵심 미너비니 기술 조건 6개 중 4개 이상 통과했는가
- 시가총액 기준을 통과했는가
- 50일 평균 거래대금 기준을 통과했는가
- 최소 주가 기준을 통과했는가

### `/미너비니 사후검증`

과거에 추천된 후보가 이후 실제로 괜찮았는지 확인합니다.

추천 당시 가격과 현재 가격을 비교해 수익률, 경과 일수, 성공/관찰/실패 판정을 보여줍니다.

```text
/미너비니 사후검증
/미너비니 사후검증 티커:NVDA
/미너비니 사후검증 목표수익률:0.15 실패수익률:-0.08
```

기본 판정 기준:

- 성공: 추천가 대비 +10% 이상
- 실패: 추천가 대비 -8% 이하
- 관찰: 성공과 실패 사이

이 기준은 옵션으로 바꿀 수 있습니다.

## 종목 추천 방식

현재 MVP는 정식 거래소 전체 스크리너가 아니라, `.env`의 `DEFAULT_TICKERS`에 넣어둔 후보군을 대상으로 계산합니다.

봇이 하는 일은 다음과 같습니다.

1. `DEFAULT_TICKERS`의 모든 티커를 순서대로 가져옵니다.
2. 각 종목의 1년 가격 데이터, 거래량, 시가총액, 재무 보조 지표를 확인합니다.
3. 미너비니식 기술 조건과 유동성 조건을 점수화합니다.
4. 검증 기준을 통과한 종목만 남깁니다.
5. 점수가 높은 순서대로 디스코드에 보여줍니다.
6. 추천 당시 가격, 점수, 등급, 추천 시간을 로컬 기록으로 저장합니다.
7. 이후 `/미너비니 사후검증`으로 실제 성과를 확인합니다.

후보군을 넓히고 싶으면 `.env`의 `DEFAULT_TICKERS`를 더 많이 넣으면 됩니다. 향후에는 S&P 500, Nasdaq 100, Russell 1000 같은 전체 유니버스를 자동으로 불러오도록 확장할 수 있습니다.

## 분석 기준

### 1. 미너비니식 기술 조건

- 현재가가 150일/200일 이동평균선 위에 있는지
- 50일 이동평균선이 150일/200일 이동평균선 위에 있는지
- 200일 이동평균선이 상승 중인지
- 주가가 52주 고점 근처인지
- 주가가 52주 저점 대비 충분히 회복했는지
- 거래량이 최근 평균보다 증가했는지

### 2. 시가총액 및 거래대금 필터

기본값은 다음과 같습니다.

```text
MIN_MARKET_CAP_USD=2000000000
MIN_AVG_DOLLAR_VOLUME_USD=20000000
MIN_PRICE_USD=10
```

기본 기준:

- 시가총액 20억 달러 이상
- 50일 평균 거래대금 2천만 달러 이상
- 주가 10달러 이상

더 보수적으로 운용하려면 다음처럼 올릴 수 있습니다.

```text
MIN_MARKET_CAP_USD=5000000000
MIN_AVG_DOLLAR_VOLUME_USD=50000000
MIN_PRICE_USD=10
```

### 3. 재무 품질 보조 지표

현재 MVP에서는 `yfinance`로 가져올 수 있는 범위에서 다음 항목을 확인합니다.

- 매출 성장 여부
- 영업이익률
- 총자산 대비 부채 부담
- 후행 PER 및 선행 PER

## 설치 방법

Python 3.10 이상을 권장합니다.

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

`.env.example`을 참고해 `.env` 파일을 만듭니다.

```powershell
copy .env.example .env
```

`.env`에 디스코드 봇 토큰을 넣습니다.

```text
DISCORD_BOT_TOKEN=YOUR_DISCORD_BOT_TOKEN
DISCORD_GUILD_ID=
DEFAULT_TICKERS=NVDA,AAPL,MSFT,AMZN,META,GOOGL,AVGO,TSLA,AMD,CRM,NFLX,COST,LLY,ORCL,PLTR,CRWD,NET,DDOG,SHOP,UBER
MIN_MARKET_CAP_USD=2000000000
MIN_AVG_DOLLAR_VOLUME_USD=20000000
MIN_PRICE_USD=10
```

`DISCORD_GUILD_ID`는 비워둘 수 있습니다. 다만 비워두면 슬래시 명령어가 디스코드에 반영되기까지 시간이 걸릴 수 있습니다. 빠르게 테스트하려면 디스코드 서버 ID를 넣는 것이 좋습니다.

## 디스코드 봇 생성 및 초대

1. [Discord Developer Portal](https://discord.com/developers/applications)에 접속합니다.
2. `New Application`을 생성합니다.
3. 왼쪽 `Bot` 메뉴에서 봇을 생성합니다.
4. 봇 토큰을 발급받아 `.env`의 `DISCORD_BOT_TOKEN`에 넣습니다.
5. `OAuth2` > `URL Generator`로 이동합니다.
6. Scopes에서 `bot`, `applications.commands`를 선택합니다.
7. Bot Permissions에서 다음 권한을 선택합니다.
   - Send Messages
   - Attach Files
   - Embed Links
   - Use Slash Commands
8. 생성된 초대 URL로 봇을 서버에 초대합니다.

## 실행 방법

```powershell
python -m src.bot
```

봇이 정상 실행되면 디스코드 서버에서 다음 명령어를 사용할 수 있습니다.

```text
/미너비니 종목추천
/미너비니 진단 티커:NVDA
/미너비니 검증
/미너비니 사후검증
```

## 예시 출력

```text
미너비니 추천 사후 검증

평가 대상 3개 / 성공 1개 / 관찰 2개 / 실패 0개

- NVDA: 성공 (+12.4%, 35일 경과)
  추천가 $123.45 → 현재가 $138.76 / 추천 당시 76점 B+

판정 기준은 기본적으로 +10% 이상 성공, -8% 이하 실패, 그 사이는 관찰입니다.
```

## 보안 주의사항

디스코드 봇 토큰은 비밀번호와 같습니다. 토큰이 외부에 노출되면 즉시 Discord Developer Portal에서 `Reset Token`으로 새 토큰을 발급받아야 합니다.

`.env` 파일과 `data/recommendations.json`은 `.gitignore`에 포함되어 있으므로 GitHub에 올리지 않습니다. 대신 `.env.example`만 공유합니다.

## 한계와 확장 방향

현재 버전은 MVP입니다. 데이터는 주로 `yfinance`에 의존하므로, 실서비스나 고신뢰 분석용으로는 정식 시장 데이터 API와 SEC EDGAR/XBRL 연결이 필요합니다.

확장 후보:

- S&P 500 또는 Nasdaq 100 전체 자동 스캔
- 추천 기록의 벤치마크 대비 성과 비교
- 추천 이후 최대 낙폭, 최고 수익률, 보유 기간별 성과 분석
- SEC EDGAR 기반 재무제표 직접 수집
- 영업현금흐름, 매출채권, 재고자산 분석
- 업종 대비 상대강도 계산
- 매일 장 마감 후 자동 리포트 발송
- OpenAI API를 활용한 설명형 리포트 생성
